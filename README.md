# 🚀 Astralis: Beyond Earth

### AI-Powered Astronomy Learning & Research Assistant

Astralis: Beyond Earth is an AI-powered platform that makes astronomy learning and space research more accessible. Users can explore fascinating space topics, ask questions in natural language, and receive intelligent responses generated using Google's Gemini AI.

Developed for the **Google × Kaggle Kaggiculture Hackathon**, Astralis combines modern web technologies with Retrieval-Augmented Generation (RAG) to create an engaging educational experience.

---

# 🌌 Problem Statement

Astronomy is one of the most fascinating fields of science, but reliable information is often scattered across research papers, articles, and textbooks.

Students and enthusiasts struggle to:
- Understand complex astronomical concepts.
- Search through lengthy research papers.
- Get quick, reliable explanations.
- Learn interactively.

Astralis addresses these challenges through an AI-powered conversational interface.

# 🚀 How It Works

1. User opens Astralis.
2. Explores astronomy topics.
3. Reads definitions and interesting facts.
4. Asks questions to the AI assistant.
5. Gemini generates intelligent responses.
6. RAG retrieves relevant research context for better answers.
